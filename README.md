# To-Do App

A simple and interactive To-Do application that helps users manage daily tasks efficiently. Users can add tasks, mark them as completed, and delete them. Built as a lightweight web project to practice web development skills.

---

## 🛠️ Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive and user-friendly interface
- Stores tasks locally (no backend needed)
