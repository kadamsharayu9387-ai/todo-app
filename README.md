# To-Do App (Java)

A simple and interactive **To-Do application** built in Java that helps users manage daily tasks efficiently. Users can add tasks, mark them as completed, and delete them. Built as a desktop application to practice Java programming and GUI development.

---

## 🛠️ Features

- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Stores tasks locally (in memory or file)  
- User-friendly graphical interface  

---

## 💻 Technology Used

- **Java** – Core programming language  
- **Java Swing / JavaFX** – For creating the GUI  
- **File I/O (optional)** – For storing tasks locally
